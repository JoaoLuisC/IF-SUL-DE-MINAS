Como foi feita a conversão?

Neste projeto, as conversões entre as diferentes bases numéricas (Decimal, Binário, Hexadecimal e Octal) foram implementadas manualmente, sem o uso de bibliotecas ou funções nativas que realizam essas conversões automaticamente.

Para cada conversão, foram desenvolvidos algoritmos que simulam os processos matemáticos envolvidos, como divisão sucessiva para conversões de decimal para outras bases e somas ponderadas para conversões para decimal. Abaixo, explico como foi feita cada conversão:

  • Decimal para outras bases: O número decimal é repetidamente dividido pela base alvo (2, 8 ou 16), armazenando os restos e formando o resultado final a partir do último resto para o primeiro.
  • Binário para Decimal: Cada dígito do número binário é multiplicado pela potência de 2 correspondente à sua posição, somando todos os resultados para obter o decimal.
  • Hexadecimal para Decimal: Similar ao binário, cada dígito hexadecimal é convertido para seu valor decimal equivalente, multiplicado por potências de 16 e somado para formar o número final.
  • Octal para Decimal: Assim como nas outras bases, os dígitos do octal são multiplicados por potências de 8 e somados.

Essas conversões foram feitas manualmente em JavaScript, com cada base numérica tendo uma função dedicada para garantir a correção e adequação ao propósito da atividade. O modal foi usado para exibir os resultados de forma clara, separando cada conversão em blocos distintos.
