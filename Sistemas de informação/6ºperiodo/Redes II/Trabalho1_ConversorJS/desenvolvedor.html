Como foi feita a conversão?

Para cada base numérica, utilizamos as funções de JavaScript que permitem a conversão de números:

  • Decimal: Utilizamos o método toString(base) para converter um número decimal para as bases binária, hexadecimal e octal.
  • Binário: Usamos parseInt(valor, 2) para converter o binário para decimal e depois os métodos toString(16) e toString(8) para hexadecimal e octal.
  • Hexadecimal: O hexadecimal é convertido para decimal usando parseInt(valor, 16), e em seguida convertido para binário e octal usando toString(2) e toString(8).
  • Octal: O octal é convertido para decimal com parseInt(valor, 8) e convertido para binário e hexadecimal com toString(2) e toString(16).

Essas conversões foram implementadas em um arquivo JavaScript que controla as interações com o modal para mostrar os resultados de forma organizada.
